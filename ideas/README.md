# Ideas

Store early sandbox ideas here before they become active project work.

## What Belongs Here
- rough concepts
- small experiments worth tracking
- ideas that need more shape before real work starts

## Ideas vs Projects
Ideas stay lightweight. Active projects move into `projects/` when they need clearer scope, their own notes, or a working folder.

## When To Move An Idea
Move an idea into `projects/` when you are ready to work on it beyond a quick note or backlog entry.

## Framework Note
This folder is framework/setup guidance only and not a real project by itself.
