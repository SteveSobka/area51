# Area51

Area51 is the control repo for reusable AI work patterns, prompts, handoffs, templates, and project tracking.

## Core Folders
- todo/ = shared task tracker using Active / Next / Blocked / Done
- ideas/ = idea backlog before promotion to active work
- handoffs/ = handoff packets and templates
- projects/ = project-specific workspaces
- templates/ = reusable templates
- scripts/ = helper scripts
- chatgpt/ = ChatGPT-specific working material

## Working Rules
- Keep tasks synchronized in todo/TODO.md
- Work in tiny practical steps
- Prefer simple, reviewable changes
- Promote ideas from ideas/ into projects/ only when ready
