# Scripts

Store small practical helper scripts here.

## What Belongs Here
- one-off helpers worth keeping
- setup or cleanup scripts
- small utilities used during sandbox work

## How To Describe A Script
For each script, add a short note near the file or in the script header that covers:
- purpose
- inputs
- outputs
- risks
- next step

## Keep It Simple
- prefer scripts that are easy to run and review
- avoid turning this folder into a full toolchain
